globalAlpha

Устанавливает/возвращает текущее значение прозрачности или альфа-канала графического объекта.

Синтаксис

context.globalAlpha=value

value - значение прозрачности. Должно быть число между 0.0 (полная прозрачность) и 1.0 (нет прозрачности).

Пример

var c = document.getElementById('myCanvas');
var ctx = c.getContext('2d');
ctx.fillStyle = 'red';
ctx.fillRect(20, 20, 75, 50);

// Включим прозрачность
ctx.globalAlpha = 0.2;
ctx.fillStyle = 'blue';
ctx.fillRect(50, 50, 75, 50);
ctx.fillStyle = 'green';
ctx.fillRect(80, 80, 75, 50);
