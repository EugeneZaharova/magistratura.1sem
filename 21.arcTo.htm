arcTo

Создает дугу/кривую между двумя касательными.

Синтаксис

context.arcTo(x1, y1, x2, y2, r)

Параметры

  • x1 - координата х первой касательной;
  • y1 - координата y первой касательной;
  • x2 - координата х второй касательной;
  • y2 - координата y второй касательной;
  • r - радиус дуги.

Пример

var c = document.getElementById('myCanvas');
var ctx = c.getContext('2d');
ctx.beginPath();
// Создаем начальную точку
ctx.moveTo(20, 20);
// Создаем горизонтальную линию
ctx.lineTo(100, 20);
// Создаем дугу
ctx.arcTo(150, 20, 150, 70, 50);
// Продолжаем линию по вертикали
ctx.lineTo(150, 120);
// Рисуем все
ctx.stroke();
