filter

Добавляет эффекты фильтра.

Синтаксис

ctx.filter = "<filter-function1> [<filter-function2>] [<filter-functionN>]";
ctx.filter = "none";

Параметры

  • blur() - размытие по Гауссу;
  • brightness() - изменение яркости;
  • contrast() - контраст рисунка;
  • drop-shadow() - эффект тени;
  • grayscale() - рисунок в оттенках серого;
  • hue-rotate() - поворот оттенка на чертеже;
  • invert() - инверсия цвета;
  • opacity() - прозрачность;
  • saturate() - насыщенность;
  • sepia() - эффект сепии;
  • none - начальное значение без фильтров.

Пример

const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

ctx.filter = 'blur(4px)';
ctx.font = '48px serif';
ctx.fillText('Hello world', 50, 100);
