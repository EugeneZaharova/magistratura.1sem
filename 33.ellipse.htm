ellipse

Добавляет эллиптическую дугу.

Синтаксис

ctx.ellipse(x, y, radiusX, radiusY, rotation, startAngle, endAngle [, anticlockwise])

value - значение прозрачности. Должно быть число между 0.0 (полная прозрачность) и 1.0 (нет прозрачности).

Параметры

  • x - координата х центра эллипса;
  • y - координата y центра эллипса;
  • radiusX - радиус большой оси эллипса; должен быть неотрицательным;
  • radiusY - радиус малой оси эллипса; должен быть неотрицательным;
  • rotation - поворот эллипса в радианах;
  • startAngle - угол, под которым начинается эллипс, измеряется по часовой стрелке от положительной оси x и выражается в радианах;
  • endAngle - угол, под которым заканчивается эллипс, измеряется по часовой стрелке от положительной оси x и выражается в радианах;
  • anticlockwise - необязательный атрибут; рисует эллипс против часовой стрелки (true); значение по умолчанию - false.

Пример

const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

// Рисует эллипс
ctx.beginPath();
ctx.ellipse(100, 100, 50, 75, Math.PI / 4, 0, 2 * Math.PI);
ctx.stroke();

// Рисует линию отражения эллипса
ctx.beginPath();
ctx.setLineDash([5, 5]);
ctx.moveTo(0, 200);
ctx.lineTo(200, 0);
ctx.stroke();
