resetTransform

Сбрасывает текущее преобразование к единичной матрице.

Синтаксис

void ctx.resetTransform()

Пример

const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

// Рисует повернутый прямоугольник
ctx.rotate(45 * Math.PI / 180);
ctx.fillRect(60, 0, 100, 30);

// Сброс матрицы преобразования к единичной матрице
ctx.resetTransform();
