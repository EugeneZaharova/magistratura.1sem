setLineDash

Задает шаблон пунктирной линии.

Синтаксис

ctx.setLineDash(segments)

Параметры

  • segments - числа, которые определяют расстояния, на которых рисуются пунктиры.

Пример

const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

// Пунктирная линия
ctx.beginPath();
ctx.setLineDash([5, 15]);
ctx.moveTo(0, 50);
ctx.lineTo(300, 50);
ctx.stroke();

// Сплошная линия
ctx.beginPath();
ctx.setLineDash([]);
ctx.moveTo(0, 100);
ctx.lineTo(300, 100);
ctx.stroke();
